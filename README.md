# AndroidCustomWidget
该项目主要是针对书籍 《Android 自定义控件开发入门与实践》这本书写的测试代码。其中 app module 是根据书中出现的小知识点的顺序，手写的测试用例。其余 module 来自本书提供的备份代码，通过自己的消化和理解，加上注释，和完善后的代码。本书可以在这里提交错误记录 -- [勘误网址](http://www.broadview.com.cn/book/93)。
